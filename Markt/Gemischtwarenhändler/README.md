# Gemischtwarenhaendler

Eigenstaendige Marktseite fuer einfache Alltags- und Reisegueter: Ausruestung, Werkzeug, Lichtquellen, Schreibmaterial, Hygiene, Kleidung, Proviant, Geschirr und Handwerkskram.

## Dateien

- `index.html` enthaelt die Seitenstruktur.
- `styles/gemischtwaren.css` enthaelt Layout, Farben, Header, Karten, Modal- und Vergleichsansicht.
- `data/goods-data.js` enthaelt Waren, Preise, Beschreibungen, Kategorien und Profilwerte.
- `scripts/gemischtwaren.js` rendert Suche, Filter, Details, Vergleich und Diagramme.

## Warenprofil

Alle Waren nutzen sechs Werte:

- Nutzen
- Haltbarkeit
- Tragbarkeit
- Alltag
- Reise
- Preiswert

Die Preise werden in Kupfertalern gefuehrt.
