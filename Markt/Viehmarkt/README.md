# Viehmarkt

Eigenstaendige statische Marktseite fuer Nutztiere, Hofvieh, Lasttiere und besondere Zuchtlinien.

## Dateien

- `index.html` enthaelt die Seitenstruktur.
- `styles/viehmarkt.css` enthaelt Layout und Gestaltung.
- `scripts/viehmarkt.js` enthaelt Filter, Detaildialoge, Vergleich und Profil-Diagramme.
- `data/livestock-data.js` enthaelt Tiere, Preise, Bilder, Unterhalt und Profilwerte.

Neue Tiere werden in `data/livestock-data.js` in `window.LIVESTOCK_DATA.animals` ergaenzt.
