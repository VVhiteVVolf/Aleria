# CharakterDatenbank

Such- und Importbereich fuer Charakterdaten. Dieser Ordner ist bewusst von einem spaeteren Ordner `Charaktere/` getrennt, in dem einzelne Steckbriefe und Biografien liegen koennen.

Firebase Collection:

- `characterDatabase`

Lokale Quelldaten koennen in vielen kleinen JSON-Dateien unter `data/` gepflegt werden. In Firebase landen die Charaktere trotzdem gemeinsam in einer Collection, damit globale Suche und Filter funktionieren.
